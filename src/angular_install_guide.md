#Comandos de instalacion para ANGULAR
##Crear proyecto: 
```
ng new my-project

```
##Lanzar servidor de desarrollo: 
```javascript
ng serve
```
##Lanzar servidor de desarrollo y que abra el navegador autom√°ticamente:
```javascript

```
##Lanzar el servidor en un puerto especifico: 
```javascript

```
